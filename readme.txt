Ваш комп не тянет хром, но на нём есть третий питон? Тогда этот POP3 клиент для вас!

Выполните python client.py -c, и работайте со своим яндексовским почтовым ящиком в интерактивном режиме.
Чтобы подключиться к ящику от другого производителя, посмотрите справку в client.py --help и укажите нужный сервер.

Поддерживаются основные команды протокола POP3:
    AUTH, USER, DELE, PASS, STAT, LIST, TOP, RSET, RETR, NOOP, QUIT.
Для подробной информации о команде CMD напишите help CMD.
О типичной POP3 сессии читайте на Вики: https://ru.wikipedia.org/wiki/POP3#Пример_сессии
